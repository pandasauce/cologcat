# CoLogCat
Colorized logcat output


## Output
![ScreenShot](https://raw.githubusercontent.com/yafp/cologcat/master/images/screenshots/cologcat_example_output.png)


## Requirements
* adb 
* whiptail (optional)

## Usage
* Make cologcat.sh executable: `chmod +x cologcat.sh`
* Start: `./cologcat.sh`
